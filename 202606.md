

### iliyami/MacSai: Mac Sai: the open-source Mac cleaner, optimizer, and malware scanner. A free, Apple-notarized alternative to CleanMyMac, built with Swift 6 and SwiftUI.
https://github.com/iliyami/MacSai


![Mac Sai 界面截图](https://github.com/iliyami/MacSai/raw/main/assets/demo.png)
Mac Sai 是开源的免费 macOS 工具，替代 CleanMyMac，提供一体化清洁、优化和恶意软件扫描功能。基于 Swift 6 和 SwiftUI 开发，支持简体双语切换。核心功能包括智能一键扫描涵盖 16 类系统垃圾（如缓存、日志）、恶意软件检测三重深度扫描、应用程序卸载自动关联残留文件、磁盘树状图空间可视化、性能优化工具（维护脚本、登录项管理）及菜单栏系统监控实时视图。采用安全优先设计：XPC 特权隔离、路径保护白名单、苹果公证保障无篡改，且无订阅、遥测与广告。通过 Homebrew 一键安装或下载 DMG 部署，兼容 macOS 14+，开源协议为 BSD-3-Clause，支持社区贡献。



### hugohe3/ppt-master
https://github.com/hugohe3/ppt-master


![PPT Master Preview](https://github.com/hugohe3/ppt-master/raw/main/docs/assets/screenshots/preview_pritzker_2026.png)

PPT Master 是一款开源 AI 工具，基于 Python 开发，能够从任意文档（如 PDF、DOCX）生成原生可编辑的 PowerPoint (.pptx) 文件。核心优势在于输出文件完全基于 PowerPoint 绘图元素（DrawingML），用户可直接在 PowerPoint 中修改所有文本、图表和动画，而非生成不可编辑的图片堆叠。项目推荐使用 Claude Opus 或 Gemini 搭配 gpt-image-2 模型实现最佳效果，支持本地处理保障数据隐私。

安装仅需 Python 环境及 pip 依赖，提供 Windows、macOS/Linux 详细指南。用户可通过 AI IDE（如 Claude Code 或 VS Code Copilot）提交文档，工具自动分析内容并输出包含过渡效果、演讲笔记的原生 PPTX 文件。集成 PackyCode、APIKEY.FUN 等赞助商 API，低成本接入多模型服务。项目开源免费，MIT 许可，强调人工精修必要性，提供六大风格示例模板下载参考。
