

### debugtheworldbot/keyStats
https://github.com/debugtheworldbot/keyStats


![KeyStats概览](https://github.com/user-attachments/assets/c5146f66-0aa3-49ce-9184-2c75301bf398)
KeyStats 是一款轻量级的 macOS 菜单栏应用，专用于实时统计键鼠使用数据。它监控键盘敲击总次数、鼠标左键和右键点击量、鼠标移动距离以及页面滚动累计距离，核心指标直接显示在菜单栏（例如键盘和鼠标点击总数）。应用在每天午夜自动重置数据，但支持手动清零，并确保重启后数据本地持久化。首次运行需授予辅助功能权限以监听全局事件，系统要求 macOS 13.0 或更高版本。基于 Swift 5.0 和 AppKit 框架开发，KeyStats 仅统计使用次数，不记录具体按键内容或位置，保障用户隐私安全。数据可通过点击图标弹出的详细面板查看，支持从 GitHub Release 直接下载安装。



### ddlmanus/MacOptimizer
https://github.com/ddlmanus/MacOptimizer


![MacOptimizer](https://github.com/ddlmanus/MacOptimizer/raw/main/image.png)  
MacOptimizer 是一款专为 macOS 13.0+设计的系统优化工具，提供八大核心功能：包括中英双语切换的系统监控（实时显示CPU、内存、磁盘状态）、应用卸载（智能扫描残留文件）、垃圾清理（系统/浏览器缓存）、系统加速（启动项管理）、大文件查找、废纸篓管理、深度清理（孤立文件识别）和文件浏览器（支持终端集成）。支持Apple Silicon与Intel芯片，可通过Homebrew安装或源码编译（Swift 5.9/SwiftUI 4.0构建）。包含安全机制如"移至废纸篓"恢复功能，并建议操作前备份数据。当前版本2.2.0，遵循MIT开源协议。



### xxnuo/MTranServer
https://github.com/xxnuo/MTranServer


![MTranServer](https://raw.githubusercontent.com/xxnuo/MTranServer/main/images/preview.png)MTranServer 是一个专注于离线翻译的高性能模型服务器，无需显卡支持即可实现超低资源消耗和极快响应速度，单个请求平均响应时间为 50 毫秒。它支持全球主要语言的互译，支持 Windows/Mac/Linux 跨平台一键部署，提供桌面端图形界面和 Docker Compose 部署方案。项目强调本地离线运行模式实现无限免费翻译，但其翻译精度低于在线大型模型。

通过兼容接口，MTranServer 支持对接多款主流翻译插件，如沉浸式翻译、简约翻译，并提供 DeepL/Google API 的替代方案。其生态包含 MTranCode IDE 注释翻译插件和正在开发的浏览器扩展，用户可通过命令参数调控日志级别、端口等设置。与其他离线项目相比，MTranServer 在内存占用、并发性能和响应速度方面表现突出，但翻译效果处于中等水平。v4 版本优化了内存使用并增强稳定性。
