

### frankovo/dns-benchmark-tool
https://github.com/frankovo/dns-benchmark-tool


![DNS Benchmark Tool](https://raw.githubusercontent.com/frankovo/dns-benchmark-tool/main/docs/real_time_tracking.gif)  
DNS Benchmark Tool 是一款高性能 DNS 测试工具，隶属于 BuildTools 网络性能套件，支持 DNSSEC 验证、DoH/DoT 加密协议以及企业级功能。通过 Python 安装（`pip install dns-benchmark-tool`），用户可使用新增的三大 CLI 命令：  
1. **top**：快速按速度和可靠性排序解析器  
2. **compare**：并排对比解析器性能并导出 CSV/Excel  
3. **monitoring**：持续监控性能，支持阈值告警  
工具提供异步查询、多轮测试和统计学分析（P95/P99延迟、抖动率），输出支持 CSV/Excel/PDF 格式。社区一周下载量超 1400 次，经 Hacker News 推广后用户激增，典型场景包括开发者优化 API 延迟、DevOps 验证迁移方案及网络管理员自动化健康检查。企业功能（如 TSIG 认证 / 区域传输）即将推出。



### Anionex/banana-slides
https://github.com/Anionex/banana-slides


![香蕉幻灯片应用](https://github.com/user-attachments/assets/6f9e4cf9-912d-4faa-9d37-54fb676f547e)
Banana Slides是一款基于nano banana pro AI模型的PPT生成应用，旨在降低用户制作专业演示文稿的门槛。它支持三种创作路径：用户输入想法、大纲或页面描述，即可自动生成完整PPT文稿。关键功能包括素材解析（支持PDF/Docx等文件上传自然语言编辑）、口头指令修改内容和区域重绘、导出为PPTX/PDF格式。应用采用Docker部署（推荐）或源码安装，后EndFlask+Gemini/OpenAI API实现AI能力，前端为React架构。适用场景覆盖小白用户、学生及职场人士，帮助用户快速创建设计统一的幻灯片，并计划扩展搜索、动画和多语种支持功能。项目采用CC BY-NC-SA非商业协议开源。



### rendercv/rendercv
https://github.com/rendercv/rendercv


![RenderCV 示例简历](https://raw.githubusercontent.com/rendercv/rendercv/main/docs/assets/images/classic.png)  
RenderCV 是一款面向学者和工程师的开源简历生成工具。通过 YAML 格式输入个人资料，用户可一键生成排版精美的 PDF 简历。核心功能包括：支持多个主题模板灵活切换，JSON Schema 提供交互式编辑与自动补全，严格的验证机制确保数据精准无错，多语言本地化适配（如日期格式等），以及细节设计自定义（页边距、色彩、字体等）。工具优先 Python 3.12+ 环境运行，安装命令为 `pip install "rendercv[full]"`，支持模板选择、版本控制和实时预览，用户可通过简单命令如 `rendercv new` 和 `rendercv render` 完成简历创建与格式化。



### google/langextract: A Python library for extracting structured information from unstructured text using LLMs with precise source grounding and interactive visualization.
https://github.com/google/langextract


![Romeo and Juliet Basic Visualization](https://raw.githubusercontent.com/google/langextract/main/docs/_static/romeo_juliet_basic.gif)
LangExtract 是 Google 开源的结构化信息提取 Python 库，它通过大型语言模型（LLM）处理非结构化文本。该库支持自定义提取任务和规则，核心优势包括：
1. 精确的源文本锚定功能，可高亮显示数据来源位置；
2. 基于提示工程生成可靠结构化输出；
3. 针对长文档优化，采用分块处理和多次提取优化召回率；
4. 生成交互式 HTML 可视化结果以审查提取实体；
5. 支持多种模型（如 Gemini、OpenAI）和本地 LLM（Ollama）；
6. 无需模型微调即可适配医疗报告、文学分析等领域；
7. 提供云模型 API 密钥设置和批量处理功能；
8. 通过插件系统支持自定义模型供应商集成。  
适用于医学笔记、病历分析等任务，开发者可通过 pip 安装并使用 Vertex AI 降低大规模处理成本。



### 9bingyin/Gist
https://github.com/9bingyin/Gist


Gist 是一款基于 Next.js 的现代 RSS 阅读器应用。该项目提供了完整的安装和构建指南：安装依赖使用 Bun（命令：`bun install`），初始化通过复制环境变量（`.env`）、生成 Prisma client 文件并初始化数据库。开发服务器通过 `bun run dev` 启动，生产构建则采用 `bun run build` 和 `bun run start` 命令。项目采用 GPL-2.0 开源许可协议，实现轻量高效的RSS订阅管理，适合开发者快速部署使用。
