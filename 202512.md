

### frankovo/dns-benchmark-tool
https://github.com/frankovo/dns-benchmark-tool


![DNS Benchmark Tool](https://raw.githubusercontent.com/frankovo/dns-benchmark-tool/main/docs/real_time_tracking.gif)  
DNS Benchmark Tool 是一款高性能 DNS 测试工具，隶属于 BuildTools 网络性能套件，支持 DNSSEC 验证、DoH/DoT 加密协议以及企业级功能。通过 Python 安装（`pip install dns-benchmark-tool`），用户可使用新增的三大 CLI 命令：  
1. **top**：快速按速度和可靠性排序解析器  
2. **compare**：并排对比解析器性能并导出 CSV/Excel  
3. **monitoring**：持续监控性能，支持阈值告警  
工具提供异步查询、多轮测试和统计学分析（P95/P99延迟、抖动率），输出支持 CSV/Excel/PDF 格式。社区一周下载量超 1400 次，经 Hacker News 推广后用户激增，典型场景包括开发者优化 API 延迟、DevOps 验证迁移方案及网络管理员自动化健康检查。企业功能（如 TSIG 认证 / 区域传输）即将推出。
