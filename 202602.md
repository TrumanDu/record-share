

### jsattler/BetterCapture
https://github.com/jsattler/BetterCapture


![BetterCapture Header](https://github.com/jsattler/BetterCapture/raw/main/website/public/bettercapture-header.png)

BetterCapture是一款专为macOS设计的免费开源屏幕录制工具，采用SwiftUI和ScreenCaptureKit开发，提供原生菜单栏集成。核心功能包括专业视频编码如ProRes、HEVC和H.264（支持HDR及透明通道），可同时录制系统音频与麦克风，并能过滤指定内容。应用注重隐私安全，无用户跟踪和数据收集，所有录制文件本地存储。支持通过Homebrew命令行工具安装（`brew install jsattler/tap/bettercapture`）或从GitHub Releases下载DMG文件，需运行macOS 15.2或更高版本。基于MIT许可证开源，鼓励社区贡献且已向同类型项目致谢。



### lakernote/EasyPostman
https://github.com/lakernote/EasyPostman


![Main Interface](https://github.com/lakernote/EasyPostman/raw/master/docs/home-en.png)  
EasyPostman 是一款开源 API 调试与压力测试工具，融合了 Postman 和 JMeter 的核心功能。通过 Java Swing 实现跨平台支持，可在 Windows、macOS 和 Linux 离线运行，提供隐私优先的本地存储机制，拒绝云端同步以保护用户数据。核心功能包括多协议支持（HTTP/HTTPS/WebSocket/SSE）、Git 工作区协作、多环境变量管理；性能测试模块支持线程组模式、实时监控和可视化报告。支持导入 Postman v2.1 及 curl 数据，提供中英双语界面和深浅主题切换。开发者可通过 GitHub/Gitee 下载预编译包或源码构建，项目文档涵盖功能详解、架构说明和脚本 API，鼓励社区通过报告问题、提交代码或完善文档参与贡献。
