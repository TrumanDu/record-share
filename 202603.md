

### JOYCEQL/magic-resume
https://github.com/JOYCEQL/magic-resume


![Magic Resume](https://github.com/user-attachments/assets/4667e49a-7bf2-4379-9390-725e42799dc7)  
Magic Resume 是一款基于 TanStack Start 和 Motion 构建的现代化在线简历编辑器，支持实时预览与自定义主题。核心特性包括深色模式、PDF导出、自动保存及硬盘级存储，并整合了Tiptap富文本编辑与Tailwind CSS样式。开发者可通过克隆仓库、安装依赖后运行开发服务器快速启动本地实例，也支持 Docker Compose 一键部署。技术栈涵盖 TypeScript、Zustand、Shadcn/ui 等主流工具。项目采用 Apache 2.0 协议，路线图规划中已实现 AI 辅助编写和多语言支持，未来计划扩展模板库和在线托管功能。用户可通过 GitHub 页面或 Discord 社区获取支持。



### Picture View
https://wl879.github.io/apps/picview/index.html


Picture View 是一款专为 MacOS 设计的轻量级图片浏览应用，仅有 7.8MB 大小。它能快速查看图像而不产生任何缓存，适合用作默认图像查看工具。应用提供两种浏览模式：单图模式下可像文档一样滚动图像，或多列模式对长图友好，确保清晰展示。操作便捷，包括优化鼠标滚轮切换图片、双点调整大小、长按放大以及右键菜单。实用功能涵盖置顶窗口、防止系统休眠、查看 EXIF 信息和多窗口支持。兼容 MacOS 10.14 及以上系统，能运行于苹果 M1 和 Intel 芯片，满足日常图像查看需求。



### SuperCmdLabs/SuperCmd
https://github.com/SuperCmdLabs/SuperCmd


![SuperCmd Screenshot](./assets/supercmd.png)
SuperCmd是一款开源macOS启动器应用，整合了Raycast兼容扩展、Wispr Flow语音工作流、Speechify文本朗读、AI与内存管理功能。关键特性包括Raycast扩展兼容性支持实时安装更新脚本命令、原生语音输入（按住说话释放后自动文字转换）、AI集成（支持Ollama/OpenAI/Anthropic/ElevenLabs等多个服务提供商）以及自动化助手如热键和片段扩展。安装需下载dmg文件并授权macOS的辅助功能、输入监控等权限。AI配置通过设置界面完成，支持自定义API密钥和模型选择，所有设置存储在本地JSON文件中。开发需Node.js 22+和Xcode工具，提供详细贡献指南与调试方案。



### 你不知道的 Claude Code：架构、治理与工程实践 - Tw93
https://tw93.fun/2026-03-12/claude.html


本文分享了作者深度使用Claude Code的经验，将其架构拆解为六层：CLAUDE.md（项目契约层）、Tools/MCP（动作能力层）、Skills（按需工作流层）、Hooks（强制约束层）、Subagents（隔离执行层）和Verifiers（验证闭环层）。文章剖析了核心机制如代理循环（收集→行动→验证）和上下文管理，指出MCP工具定义常挤占12.5%的Token预算致有效信息被淹没。关键实践包括：保持CLAUDE.md简短可执行（仅2-5K tokens）、Skills按需加载且禁用副作用自触发、工具名需显式分层（如github_pr_*）、Hooks拦截高风险操作、Subagents隔离上下文。作者强调验证闭环（测试/lint/截图）的必要性，并开源健康检查工具/claude-health排查配置失衡。最后指出用户需从"工具使用"转向"系统设计"，确保任务可验可审计方能高效自治。

### pbakaus/impeccable
https://github.com/pbakaus/impeccable


Impeccable是一个由Paul Bakaus创建的AI增强前端设计工具，旨在解决大型语言模型（LLMs）常见的审美偏见问题，如过度使用Inter字体或嵌套卡片等。其核心基于Anthropic的前端设计技能，扩展了7个专业参考模块覆盖排版、颜色、空间设计等领域；提供20个命令如`/audit`进行技术审查、`/polish`实现最终优化、`/animate`添加动态效果；并强制避开反模式如灰色文字背景或弹性动效。支持主流开发工具包括Cursor、VS Code Copilot和Gemini CLI，安装可通过官网下载或文件复制，使用命令直接优化目标元素。开源项目采用Apache 2.0许可。
