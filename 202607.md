

### assistant-ui/assistant-ui
https://github.com/assistant-ui/assistant-ui


![assistant-ui](https://raw.githubusercontent.com/assistant-ui/assistant-ui/main/.github/assets/header.svg)

assistant-ui 是一个开源的 TypeScript/React 库，用于快速构建生产级 AI 聊天界面，类似 ChatGPT 的用户体验。它提供可组合组件如 Thread、Message 和 Composer，支持流式传输、语音输入、Markdown 渲染等生产功能，并集成多种后端服务包括 Vercel AI SDK、LangChain 和 Google AI SDK，兼容 OpenAI、Gemini 等主流模型。开发者可通过 CLI 或 npm 安装，灵活定制界面（如实现 Perplexity 风格），并通过类型安全 API 确保开发效率。已被 Mastra、LangChain 等公司用于生产环境，提供文档、示例和社区支持，支持 React Native、终端跨平台开发，采用 MIT 许可证可免费使用。



### nexmoe/VidBee
https://github.com/nexmoe/VidBee


![VidBee 主界面](https://raw.githubusercontent.com/nexmoe/VidBee/master/screenshots/main-interface.png)  
VidBee 是一款基于 Electron 和 yt-dlp 的开源视频下载工具，支持从 YouTube、TikTok、Instagram 等全球 1000 多个网站下载视频和音频。它提供直观的现代界面，功能包括实时进度跟踪、下载队列管理及一键暂停/恢复操作。核心特性为 RSS 自动订阅下载，可后台监控创作者更新并自动获取新内容。用户可自定义输出格式（如 MP4、MKV 或原始格式），工具还包含 Web API 和 Docker 部署选项，支持社区贡献开发。项目处于活跃维护阶段，可通过 Github 免费下载使用。



### browser-act/skills
https://github.com/browser-act/skills


BrowserAct Skills 是一款专为AI代理设计的浏览器自动化命令行工具（CLI），旨在解决反机器人检测问题并优化代理操作。核心功能包括：突破防爬墙机制的三层防御（环境层指纹欺骗、执行层自动验证码破解、人类层无缝远程接管）；支持三种浏览器模式（Chrome复用登录状态、隐私模式新指纹、固定身份模式）；实现并发无干扰任务（独立浏览器隔离账户隐私，多会话共享状态）。优化AI交互特性：如紧凑文本输出减少Token消耗、索引式操作简化文档解析，并内置语义会话记忆。免费版本支持基本Chrome模式和5个以下隐身浏览器，付费解锁更多隐身实例和代理服务。额外提供Skill Forge工具，可自动化生成网站爬虫解决方案，预置30+平台数据提取技能（如LinkedIn/YouTube），兼容主流开发环境与代理工具。安装命令简单，涵盖Windows/macOS/Linux系统。



### wiltodelta/remove-ai-watermarks: AI watermark remover. CLI and Python library to strip visible and invisible AI watermarks (Gemini / Nano Banana sparkle, SynthID) and provenance metadata (C2PA, EXIF, IPTC) from images.
https://github.com/wiltodelta/remove-ai-watermarks


![Remove AI Watermarks](https://raw.githubusercontent.com/wiltodelta/remove-ai-watermarks/main/demo_banana_before.png)

Remove-AI-Watermarks 是一款开源工具，可一键移除AI生成图像中的可见与不可见水印及元数据。支持Google Gemini（火花徽标）、OpenAI DALL-E（SynthID）、Stable Diffusion、字节跳动豆包／即梦文字水印（如"豆包AI生成"）等多种平台水印。  
**核心功能**：  
1. **可见水印移除**：通过反alpha混合算法还原原始像素（适配Gemini火花、Doubao中文标签等），CPU处理速度约0.05秒/图。  
2. **不可见水印清除**：基于扩散模型（如SDXL）重构图像消除SynthID等数字水印，需GPU支持。  
3. **元数据剥离**：清除C2PA凭证、EXIF "Made with AI"标签等AI标识元数据。  
提供在线服务[raiw.cc](https://raiw.cc)，无需本地配置。支持批量处理与ComfyUI集成，强调合法使用场景（开源协议MIT）。工具通过命令行操作，检测阶段可自动识别水印来源。



### facebook/astryx
https://github.com/facebook/astryx


Astryx 是一个开源设计系统，最初在 Meta 内部开发八年，现已开放 Beta 版本测试。该系统基于 React 和 StyleX 构建，提供 150 多个可访问组件、自定义主题（含七种预设如中性/巧克力等）、暗黑模式、模板和 CLI 工具。核心特点包括：
- **开放式架构**：组件支持深度组合与源码修改（swizzle 功能）。
- **无样式锁定**：可直接使用 `className` 覆盖样式，兼容 Tailwind、CSS 模块等工具。
- **无缝主题定制**：通过 CSS 变量实现主题覆盖，无需修改核心代码。
- **AI 协同设计**：API 和 CLI 均优化为人与 AI 助手协作开发。
安装需通过 npm/pnpm/yarn 添加 `@astryxdesign/core` 核心包及主题包，并集成 CLI 脚本。系统严格遵循文档约定、测试驱动原则，为人类和 AI 提供一致的开发体验。支持 Node 22+ 和 pnpm 11，采用 MIT 协议开源。



### Taste — Design DNA for Agents
https://www.tastelab.xyz/


![Taste Lab](https://www.tastelab.xyz/avatar.png)

Taste Lab 是一个AI驱动的工具，专注于从任何网站URL提取设计DNA，为AI代理提供完整设计上下文。该工具通过四个AI代理角色分步执行：首先提取精确设计测量（如颜色、间距、阴影值），然后检测系统性模式，推断设计品味（包括触发点、决策原因和权衡），最后过滤输出。结果生成Markdown和JSON文件，包含Design Map（完整设计 token 集）和Taste DNA（至少4条原则解析设计逻辑）。工具支持集成主流的AI开发环境如Cursor、Claude Code和GitHub Copilot，用户可克隆仓库后通过命令行运行，无需复杂设置。



### yetone/kill-ai-slop
https://github.com/yetone/kill-ai-slop


Kill AI Slop 项目致力于消除AI生成内容的视觉缺陷，包括靛蓝渐变、发光卡片和滥用emoji等常见问题。该项目分为两大核心部分：一是[网站](killaislop.com)，作为多语言指南展示32种AI设计缺陷（Catalogue of Tells），每个案例提供交互式前后对比演示；二是`kill-ai-slop` Agent Skill工具，支持扫描网页项目代码、诊断问题点并提出修复方案。用户可通过npm命令安装该工具，或运行命令行脚本解析项目结构生成报告。项目源码公开于GitHub，网站基于Astro框架静态构建无JS默认加载，所有演示案例重建为原生HTML以实现教育目的。



### Nutlope/hallmark
https://github.com/Nutlope/hallmark


![Hallmark, a design skill that refuses to look AI-generated](https://static.trumandu.top/yank-note-picgo-img-20250307214451.png)  
Hallmark 是由 Together AI 开发的设计工具，能生成避免 AI 痕迹的 UI 界面。它通过选择宏观结构、匹配 20 种主题风格、运行 57 项质量检测及自我评估，拒绝传统 AI 训练的模板化输出，确保每个设计具备独特性。工具提供四种核心功能：默认模式构建新界面；`audit` 对现有代码反模式评分；`redesign` 保留内容重建结构；`study` 提取设计 DNA 并输出便携文档。新增的 Custom 模式针对特殊需求从头定制色彩、字体和布局。项目支持通过 `npx` 安装到 Claude Code、Cursor 和 Codex 环境，采用 MIT 许可开源，提供多领域设计案例（如 SaaS 产品、时尚品牌等），所有页面均为独立 HTML+CSS 实现。
