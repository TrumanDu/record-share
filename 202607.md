

### assistant-ui/assistant-ui
https://github.com/assistant-ui/assistant-ui


![assistant-ui](https://raw.githubusercontent.com/assistant-ui/assistant-ui/main/.github/assets/header.svg)

assistant-ui 是一个开源的 TypeScript/React 库，用于快速构建生产级 AI 聊天界面，类似 ChatGPT 的用户体验。它提供可组合组件如 Thread、Message 和 Composer，支持流式传输、语音输入、Markdown 渲染等生产功能，并集成多种后端服务包括 Vercel AI SDK、LangChain 和 Google AI SDK，兼容 OpenAI、Gemini 等主流模型。开发者可通过 CLI 或 npm 安装，灵活定制界面（如实现 Perplexity 风格），并通过类型安全 API 确保开发效率。已被 Mastra、LangChain 等公司用于生产环境，提供文档、示例和社区支持，支持 React Native、终端跨平台开发，采用 MIT 许可证可免费使用。



### nexmoe/VidBee
https://github.com/nexmoe/VidBee


![VidBee 主界面](https://raw.githubusercontent.com/nexmoe/VidBee/master/screenshots/main-interface.png)  
VidBee 是一款基于 Electron 和 yt-dlp 的开源视频下载工具，支持从 YouTube、TikTok、Instagram 等全球 1000 多个网站下载视频和音频。它提供直观的现代界面，功能包括实时进度跟踪、下载队列管理及一键暂停/恢复操作。核心特性为 RSS 自动订阅下载，可后台监控创作者更新并自动获取新内容。用户可自定义输出格式（如 MP4、MKV 或原始格式），工具还包含 Web API 和 Docker 部署选项，支持社区贡献开发。项目处于活跃维护阶段，可通过 Github 免费下载使用。
